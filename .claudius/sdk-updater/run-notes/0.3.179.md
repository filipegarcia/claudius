## Summary

SDK 0.3.179 adds an optional `tool_use_meta` sidecar to assistant messages (display-friendly names for tool calls) and ships two runtime bug fixes: one preventing `-p` print-mode from exiting before a completed background agent's notification is delivered, and one fixing remote stream-json sessions incorrectly appearing busy for the full duration of a background workflow. No code changes were made. Headline risk: `tool_use_meta` will be worth revisiting once the SDK exposes a typed accessor; until then, Claudius continues to display raw wire tool names.
