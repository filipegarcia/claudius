## Summary

The 0.3.171–0.3.172 SDK range brings two changes: a new optional `skipMcpDiscovery`
field on `SdkPluginConfig` (lets an SDK host that manages a plugin's MCP connections
itself skip the engine's `.mcp.json` re-read), and a bug-fix for slash-followed-by-
whitespace input (e.g. `/ add tests`) being silently dropped instead of forwarded as
a plain prompt. Neither change requires any Claudius code modification — Claudius does
not pass a `plugins` array to `query()`, and the slash-whitespace fix is entirely
SDK-internal. The package.json version bump to `^0.3.172` is the only deliverable.
