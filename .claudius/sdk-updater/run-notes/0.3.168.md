## Summary

This is a CLI-binary-only bump. The embedded Claude Code CLI advanced from
v2.1.167 (commit `9e75a3f`, 2026-06-05) to v2.1.168 (commit `166c23a`,
2026-06-06). Every TypeScript declaration file (`sdk.d.ts`,
`agentSdkTypes.d.ts`, `sdk-tools.d.ts`, `bridge.d.ts`, `browser-sdk.d.ts`,
`assistant.d.ts`) is byte-for-byte identical between the two npm releases,
confirmed by diffing the cached Bun packages. No Claudius code changes were
required. The headline risk is any behavior introduced in the CLI binary that
is not reflected in the type surface; this cannot be caught by type-checking
and is noted under Risks.
