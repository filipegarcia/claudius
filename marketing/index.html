𐌂 Claudius

Claude Code in your browser.

Same agent. Same tools. Same sessions on disk. A web UI that gets out of the way — chat, permissions, MCP, hooks, costs, scheduled runs, all local.

curl -fsSL https://filipegarcia.gitlab.io/claudius/setup.sh | bash

Need to read it first? View the script · Source on GitLab ↗

Requires Node 20+ and git. Works on macOS and Linux. The script clones the repo into ~/claudius, runs npm install, and tells you how to start it.

Browser-native chat

Streaming text, thinking blocks, tool cards, permission modals, and the AskUserQuestion form — same shape as the TUI, no terminal.

Sessions that travel

Web session ids match the TUI's. Resume a conversation in claude --resume <id> from the same prompt you started in the browser.

Costs, schedules, files

Per-project cost roll-ups with budget caps. Cron-style scheduled runs. A workspace-scoped file tree. Inline image attachments with [Image #N] markers.

MCP, hooks, plugins

Wire up MCP servers, surface hook events as they fire, install plugins — the SDK's full surface, made clickable.

Power-user options

# Pin to a specific branch and a custom location
curl -fsSL https://filipegarcia.gitlab.io/claudius/setup.sh | bash -s -- \
  --prefix=$HOME/code/claudius \
  --branch=main

# Or download first, read, then run
curl -fsSL https://filipegarcia.gitlab.io/claudius/setup.sh -o setup.sh
less setup.sh
bash setup.sh --start    # runs `npm run dev` after install
