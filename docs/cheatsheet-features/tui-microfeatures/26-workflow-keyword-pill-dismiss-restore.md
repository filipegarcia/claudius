# Workflow keyword dismiss / restore (Alt+W, Ctrl+S stash)

**Source:** Claude Code TUI — input keyword nudge
**Status:** PARTIAL

## What it is
When the composer detects `workflow` / `workflows` / `ultraplan` / `ultrathink` / `ultrareview` in the draft, an inline pill labeled `Dynamic workflow requested for this turn` appears with an Alt+W toggle that flips the turn between active and ignored, and a `Tip: ctrl+s stash` hint suggesting you stash the prompt instead. Binary strings spell out the full contract: `workflow-keyword-active`, `Dynamic workflow requested for this turn`, ` to ignore`, `workflow-keyword-ignored`, `tengu_workflow_keyword_dismissed`, `Workflow keyword ignored for this prompt`, ` to undo`, `tengu_workflow_keyword_restored`, plus `stash-hint`, `Tip:`, `ctrl+s`, `chat:stash`. Dismissing the pill suppresses the system-reminder that would otherwise tell the model to use the Workflow tool for that turn.

## Claudius today
The pill itself exists in `components/chat/PromptInput.tsx`: `KEYWORD_HINTS` (line 103) matches `/\bworkflows?\b/i` and renders the label `Dynamic workflow requested for this turn` in the composer footer (line 1095), with a `· ignore` click affordance that adds the id to `dismissedHints` and resets on send. The comment at line 96 explicitly cites the TUI feature as the inspiration ("Claudius's take on Claude Code's 'Dynamic workflow requested for this turn · meta+w to ignore'") and notes that a browser cannot intercept ⌘W, so dismissal is by click rather than chord. The companion Dynamic Workflows session toggle (`components/panels/widgets/ModelPicker.tsx`, `app/api/sessions/[id]/ultracode/route.ts`, `lib/server/session.ts` line 1818) is what actually enables the SDK's Workflow tool. What is *not* wired: the Alt+W (or any chord) toggle, the `ultraplan` / `ultrathink` / `ultrareview` triggers in the same detector, the `workflow-keyword-ignored` → "Workflow keyword ignored for this prompt · to undo" restore state, the `tengu_workflow_keyword_dismissed` / `tengu_workflow_keyword_restored` telemetry pair, the `stash-hint` "Tip: ctrl+s stash" companion, and the system-reminder injection that nudges the model toward the Workflow tool when the pill is active.

## Decision
PARTIAL. The "Dynamic workflow requested for this turn" pill and click-to-ignore-this-draft behavior already ship in `components/chat/PromptInput.tsx`. To match the full TUI surface, the same hint row could grow a second state ("Workflow keyword ignored for this prompt · undo"), expand the regex to cover `ultraplan|ultrathink|ultrareview`, surface a sibling "Tip: ctrl+s to stash" hint (the composer already has a stashed-draft ref at line 179), and — when active — flip on the existing `ultracode` toggle for the next turn via `app/api/sessions/[id]/ultracode/route.ts` instead of (or in addition to) appending a system-reminder.
