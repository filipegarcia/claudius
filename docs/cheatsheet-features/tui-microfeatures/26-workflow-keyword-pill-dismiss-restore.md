# Workflow keyword dismiss / restore (Alt+W, Ctrl+S stash)

**Source:** Claude Code TUI — input keyword nudge
**Status:** PARTIAL

## What it is
When the composer detects `workflow` / `workflows` / `ultraplan` / `ultrathink` / `ultrareview` in the draft, an inline pill labeled `Dynamic workflow requested for this turn` appears with an Alt+W toggle that flips the turn between active and ignored, and a `Tip: ctrl+s stash` hint suggesting you stash the prompt instead. Binary strings spell out the full contract: `workflow-keyword-active`, `Dynamic workflow requested for this turn`, ` to ignore`, `workflow-keyword-ignored`, `tengu_workflow_keyword_dismissed`, `Workflow keyword ignored for this prompt`, ` to undo`, `tengu_workflow_keyword_restored`, plus `stash-hint`, `Tip:`, `ctrl+s`, `chat:stash`. Dismissing the pill suppresses the system-reminder that would otherwise tell the model to use the Workflow tool for that turn.

## Claudius today
The two-state pill ships in `components/chat/PromptInput.tsx`: `KEYWORD_HINTS` (line 127) matches `/\bworkflows?\b/i` with `label: "Dynamic workflow requested for this turn"` and `ignoredLabel: "Workflow keyword ignored for this prompt"`. The composer footer (lines 1210-1243) renders the two states — `keyword-hint-active` with `· ignore` toggles into `keyword-hint-ignored` with `· undo`, matching the TUI's `workflow-keyword-active` ↔ `workflow-keyword-ignored` flow — and the comment at line 115 explicitly cites the TUI feature as the inspiration ("Claudius's take on Claude Code's 'Dynamic workflow requested for this turn · meta+w to ignore'"), noting that a browser cannot intercept ⌘W so dismissal is by click rather than chord. A separate `ultraplan` hint already lives in the same `KEYWORD_HINTS` array (line 140) pointing at `/ultraplan`. The companion Dynamic Workflows session toggle (`components/panels/widgets/ModelPicker.tsx` line 422, `app/api/sessions/[id]/ultracode/route.ts`, `lib/server/session.ts` line 2651) is what actually enables the SDK's Workflow tool. What is *not* wired: the Alt+W (or any chord) toggle, `ultrathink` / `ultrareview` triggers (and the `workflow` pill detector folding `ultraplan` in too), the `tengu_workflow_keyword_dismissed` / `tengu_workflow_keyword_restored` telemetry pair, the `stash-hint` "Tip: ctrl+s stash" companion (the composer already has a `stashedDraftRef` at line 213 used for prompt-history recall), and the system-reminder injection that nudges the model toward the Workflow tool when the pill is active.

## Decision
PARTIAL. The "Dynamic workflow requested for this turn" pill and its two-state ignore/undo flow already ship in `components/chat/PromptInput.tsx`, as does an `ultraplan` sibling hint. To match the full TUI surface, the workflow detector could grow `ultrathink|ultrareview` triggers, surface a "Tip: ctrl+s stash" companion that reuses `stashedDraftRef`, and — when the pill is active — flip on the existing `ultracode` toggle for the next turn via `app/api/sessions/[id]/ultracode/route.ts` (or append a system-reminder server-side) instead of relying purely on the visual nudge.
