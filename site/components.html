The AnimatedGlyph

Cycling through the old Latin alphabet — running and stopped.

The little tile at the top of the workspace rail. Cycles the Old Italic script — the ancestor of the Latin alphabet, Unicode U+10300 → U+1031A — every 350 ms while the agent is running. When it stops, it snaps back to 𐌂 — Old Italic C, for Claudius. Lives in components/nav/SideNav.tsx.

useDeepLinks.ts TypeScript 
// Open a workspace from a `claudius://` URL.
export function useDeepLinks() {
  const router = useRouter();
  useEffect(() => {
    if (!isElectron()) return;
    return onDeepLink((url) => {
      const { host, pathname } = parse(url);
      if (host === "workspace") router.push(pathname);
    });
  }, [router]);
}
